# Iris Dataset - 75th Percentile Comparison

## Objective
Compare the 75th percentile (Q3) of all numeric features for each Iris species.

### Features
- Petal Length (PL)
- Petal Width (PW)
- Sepal Length (SL)
- Sepal Width (SW)

### Species
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Python Libraries
- NumPy
- Pandas

## Method Used
`numpy.percentile(array, 75)`

## Learning Outcome
- Understanding Quartiles
- 75th Percentile (Q3)
- Comparing distributions across different species
